

# SpoToolkit Mobile App

The project is a training mobile application that allows the user to look up statistics, albums, musicians, and also control the music that they're playing. Unfortunately, due to changes in Spotify's privacy policies, some data is now missing, and also using the API requires a premium account.

## Tech Stack

**Client:** Kotlin, Compose

**Server:** Spotify Api

**Additionaly:** GoogleFonts, Coolors, Android Studio, GitHub

## Usage/Examples

✔️ **RESPONSIVE** - Works well on bigger, as well as smaller devices.

✔️ **STYLISH** - True to the spotify color pallet and overall feel.

✔️ **SCALABLE** - Can be easily build upon, due to the chosen tech stack.

✔️ **GOOD READABILITY** - Usage of icons, readable fonts and contrasted colors to provide best user experience.
